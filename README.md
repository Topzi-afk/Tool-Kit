# 🚀 GAMING SYSTEM TOOLKIT v6.2 (Ultimate Edition)

**Gaming System Toolkit** — это мощный, красивый и портативный скрипт для мгновенной настройки, оптимизации и установки софта на Windows 10 и 11. Создан геймерами для геймеров. Забудь о ручном поиске программ и часах настройки системы — этот тулкит делает всё в пару кликов прямо из красивой консольной панели.

## 🌟 Особенности проекта

- **100% Open Source и бесплатно:** Никаких скрытых майнеров, платных подписок или закрытого кода. Только чистая польза.
- **Работает из коробки:** Не требует установки. Просто скачай `.bat` файл и запусти от имени Администратора.
- **Умный обход UAC:** Скрипт сам запросит права администратора (без крашей на кастомных сборках Windows).
- **Идеальная совместимость:** Полностью переписан под Windows 10 и 11 (Home и Pro). Устаревший `wmic` заменен на современный `Get-CimInstance`.
- **Адаптивный интерфейс:** Авто-настройка шрифта для больших мониторов (4K), поддержка ANSI-цветов и стильный киберпанк-дизайн.
- **Мультиязычность:** Дисклеймер и предупреждения доступны на Русском и Английском языках.
- **Безопасность:** Никаких сомнительных твиков. Только проверенные временем оптимизации реестра и служб.

## 🛠 Функционал (8 мощных разделов)

### 1. 📦 [ SOFT ] Базовый Софт и Активация
Установка самого необходимого софта одним кликом через `winget`: браузеры, архиваторы, DirectX, Visual C++, а также интеграция с **WinUtil** и **MAS** (Microsoft Activation Scripts) для легальной активации Windows и Office.

### 2. 🔬 [ DIAG ] Диагностика и Стресс-тесты
Полный набор для проверки железа: AIDA64, MSI Afterburner + RTSS, FurMark, OCCT, CPU-Z/GPU-Z, CrystalDisk и другие утилиты для мониторинга температур и стабильности.

### 3. ⚡ [ TWEAK ] Твики Windows и FPS-Оптимизация
Скрытые планы питания, отключение Xbox Game Bar/DVR, включение HAGS, отключение телеметрии, индексирования и виджетов Win 11. Инструменты для снижения DPC-латентности (ISLC, Timer Resolution).

### 4. 🌐 [ NET ] Сеть, DNS, VPN и Безопасность
Сброс сетевого стека для снижения пинга, быстрая смена DNS (Cloudflare/Google), отключение телеметрии. 
**Особенность для РФ:** Встроенное скачивание и запуск **GoodbyeDPI** (обход блокировок без VPN) и установка **AmneziaVPN** (Open Source VPN). Утилиты вроде Wireshark и Simplewall.

### 5. 🎮 [ PAD ] Геймпады и Периферия
DS4Windows, AntiMicroX, ViGEmBus, OpenRGB и другие утилиты для эмуляции геймпадов и управления подсветкой.

### 6. 🎵 [ AUDIO ] Аудио, Захват и Стриминг
OBS Studio, Streamlabs, Voicemeeter, NVIDIA Broadcast, Equalizer APO и другие инструменты для стримеров и меломанов.

### 7. 💻 [ DEV ] Разработка и Гейминг-инструменты
VS Code, Python, Node.js, JDK, Docker, Postman, Windows Terminal, а также игровые клиенты: Steam, Discord, Playnite, Heroic Games Launcher.

### 8. 🧹 [ CLEAN ] Очистка и Обслуживание
Безопасная очистка Temp/Prefetch/кэша браузеров, запуск `sfc /scannow` и `DISM`, дефрагментация, обновление всех программ через winget, очистка кэша Windows Update.

## 📥 Установка и использование

1. Скачайте файл `TopziToolkit.bat`.
2. Нажмите по нему правой кнопкой мыши и выберите **"Запуск от имени администратора"**.
3. (Опционально) Если текст кажется мелким — зажмите `CTRL` и покрутите колесико мыши вверх.
4. Выберите нужный раздел, введите цифру и нажмите `Enter`.
5. Следуйте подсказкам на экране.

> ⚠️ **РЕКОМЕНДАЦИЯ:** Перед применением глубоких твиков системы настоятельно рекомендуется создать точку восстановления Windows!
